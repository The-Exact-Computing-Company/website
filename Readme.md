# Exact Computing Website

This repository contains the static website for **The Exact Computing Company**.
The site is intentionally minimal: plain HTML, retro terminal styling, and no JavaScript.

## Development

The site is a single `index.html` file.
Open it directly in a browser to preview.

## Deployment

Serve the contents of this repository using any static file host
(e.g. Nginx, GitHub Pages, Netlify, or a simple file server).
